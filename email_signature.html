Comment installer votre signature ?

  1. Dans l'encadré blanc ci-dessous, sélectionnez tout le contenu (cliquez et glissez votre souris, ou faites Ctrl+A).
  2. Faites Copier (Ctrl+C).
  3. Allez dans les paramètres de votre boîte mail (Gmail, Outlook, etc.) à la section "Signature".
  4. Faites Coller (Ctrl+V) dans la zone de texte de la signature et enregistrez.

Note : Vous pouvez modifier les textes (Prénom, Nom, Téléphone) directement dans le code source de ce fichier ou une fois collé dans votre boîte mail.
